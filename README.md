# Sneakers Store  👟

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

**⚙️ Tech stack:**

- Vue 3
- Vue Router
- TailwindCSS
- Axios
- @formkit/auto-animate
- Composition API
- ESLint
- Prettier
- heroicons/vue
- vue-toastification
- vue-spinner
- mdb-vue-ui-kit
