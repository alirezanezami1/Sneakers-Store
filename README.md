# Sneakers Store  👟

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

#### Ready!🫴🏻


**⚙️ Tech stack:**

- Vue 3
- Vue Router
- TailwindCSS
- Axios
- @formkit/auto-animate
- Composition API / Options API
- ESLint
- Prettier
- heroicons/vue
- vue-toastification
- vue-spinner
- vue-router
- mdb-vue-ui-kit
